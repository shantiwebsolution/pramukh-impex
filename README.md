# Pramukh Impex - Nylon Coil Zippers Website

A modern, responsive one-page static website for Pramukh Impex, showcasing premium nylon coil zippers for bag and apparel manufacturers.

## Features

- Fully responsive design (mobile-first)
- Sticky navigation with smooth scrolling
- Mobile hamburger menu
- Scroll reveal animations
- Accessible FAQ accordion with keyboard support
- Back-to-top button
- Professional B2B design with blue accent colors
- Semantic HTML and ARIA attributes for accessibility

## Technologies Used

- HTML5
- CSS3 (with Grid and Flexbox)
- JavaScript (jQuery for DOM manipulation)
- Google Fonts (Roboto)

## File Structure

- `index.html` - Main HTML file
- `styles.css` - Stylesheet
- `script.js` - JavaScript functionality
- `README.md` - This file

## How to Run Locally

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. The website will load and be fully functional as a static site.

No server setup is required; it works directly from the file system.

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers
- Respects `prefers-reduced-motion` for accessibility
